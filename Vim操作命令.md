# 一、模式切换（最基础）

| 操作    | 说明          |
| ----- | ----------- |
| `i`   | 在光标前进入插入模式  |
| `a`   | 在光标后进入插入模式  |
| `o`   | 在下一行新建并进入插入 |
| `Esc` | 退出到普通模式     |
| `:`   | 进入命令行模式     |

---

# 二、光标移动（高频）

|命令|说明|
|---|---|
|`h j k l`|左 下 上 右|
|`w / b`|下一个词 / 上一个词|
|`0`|行首|
|`^`|行首第一个非空字符|
|`$`|行尾|
|`gg`|文件开头|
|`G`|文件结尾|
|`nG`|跳到第 n 行|
|`Ctrl + f / b`|向下/上翻页|

---

# 三、删除 / 复制 / 粘贴

## 删除

|命令|说明|
|---|---|
|`x`|删除当前字符|
|`dd`|删除整行|
|`ndd`|删除 n 行|
|`dw`|删除一个单词|
|`d$`|删除到行尾|
|`d0`|删除到行首|

## 复制（yank）

|命令|说明|
|---|---|
|`yy`|复制整行|
|`nyy`|复制 n 行|
|`yw`|复制一个词|
|`y$`|复制到行尾|

## 粘贴

|命令|说明|
|---|---|
|`p`|在光标后粘贴|
|`P`|在光标前粘贴|

---

# 四、撤销 / 重做

|命令|说明|
|---|---|
|`u`|撤销|
|`Ctrl + r`|重做|

---

# 五、查找与替换（开发最常用）

## 查找

|命令|说明|
|---|---|
|`/word`|向下查找|
|`?word`|向上查找|
|`n / N`|下一个 / 上一个|

## 替换

|命令|说明|
|---|---|
|`:%s/old/new/g`|全文替换|
|`:s/old/new/g`|当前行替换|
|`:%s/old/new/gc`|替换并确认|

---

# 六、保存 / 退出

|命令|说明|
|---|---|
|`:w`|保存|
|`:q`|退出|
|`:wq` 或 `ZZ`|保存并退出|
|`:q!`|强制退出不保存|

---

# 七、可视模式（选中操作）

|命令|说明|
|---|---|
|`v`|字符选择|
|`V`|行选择|
|`Ctrl + v`|块选择（列编辑很强）|
|选中后 `y d p`|复制/删除/粘贴|

---

# 八、实用高频技巧（强烈建议掌握）

| 命令                    | 说明                |
| --------------------- | ----------------- |
| `.`                   | 重复上一次操作           |
| `>> / <<`             | 缩进 / 反缩进          |
| `gg=G`                | 全文自动缩进（C/C++ 很好用） |
| `:set number`         | 显示行号              |
| `:set relativenumber` | 相对行号（开发神器）        |
| `:noh`                | 清除搜索高亮            |
