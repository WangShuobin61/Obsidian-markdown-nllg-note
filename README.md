

# 🌟 逆旅流光 - Obsidian 知识库

这是一个基于 **Obsidian** 构建的个人数字花园与知识管理系统。「逆旅流光」意喻在人生的逆旅中，把握时光的流转，将碎片化的思考凝结为永恒的知识晶体。

## 📖 简介

本仓库存储了我的第二大脑核心内容。它不仅仅是笔记的堆砌，更是一套**渐进式总结**与**双向链接**的知识网络。通过纯文本 Markdown 存储，确保数据的长期可读性与跨平台迁移能力。

### 核心特性
*   **🔗 双向链接**：构建知识图谱，发现不同领域之间的隐藏关联。
*   **🌱 数字花园**：知识持续生长、修剪与迭代，而非一成不变的静态文档。
*   **🧠 卡片盒笔记法 (Zettelkasten)**：采用原子化笔记理念，将灵感转化为永久概念。
*   **📦 本地优先**：数据完全掌控，支持端到端加密同步。
*   **🛠️ 高度定制化**：依托 Obsidian 强大的插件生态与 Minimal 主题打造。

## 🗂️ 仓库结构

知识库采用模块化的分类体系，辅以标签系统进行横向关联：

| 文件夹 / 文件 | 说明 |
| :--- | :--- |
| `01-领域知识/` | 系统性学科知识、专业技能学习笔记 |
| `02-项目笔记/` | 正在进行或已完成的项目复盘与文档 |
| `03-日常记录/` | 日记、周记、瞬间捕捉的灵感与想法 |
| `04-阅读笔记/` | 读书心得、文章摘要与知识提炼 |
| `05-资源整理/` | 工具推荐、实用模板、高频网址库 |
| `06-输出创作/` | 待整理的文章草稿、演讲稿、分享素材 |
| `07-归档备份/` | 历史版本或已完结内容的归档 |
| `_attachments/` | 笔记引用的图片、PDF 等附件资源 |
| `.obsidian/` | **核心配置目录**，包含插件、主题与工作流设置 |

## 🛠️ 技术栈与配置

本知识库经过精心配置，以下是核心依赖：

*   **📱 客户端**: [Obsidian](https://obsidian.md/) (v1.0+)
*   **🎨 主题**: [Minimal](https://github.com/kepano/obsidian-minimal) - 专注于内容阅读与写作的极简主义主题。
*   **🧩 核心插件**:
    *   `Dataview`: 数据查询与动态视图（用于 Dashboard 和高级索引）。
    *   `Templater`: 强大的模板系统，自动化笔记创建流程。
    *   `Excalidraw`: 手绘风格的图表、白板与草图绘制。
    *   `Calendar`: 日历视图集成，方便时间轴导航与日记管理。
    *   `Kanban`: 看板式项目管理，可视化任务状态。
    *   `Obsidian Git`: 版本控制，全程记录每一次思考的变更。

## 📈 知识管理理念

1.  **渐进式总结 (Progressive Summarization)**：从原始摘录，到高亮重点，再到个人思考，层层深入。
2.  **常青笔记 (Evergreen Notes)**：笔记应像常青树一样持久耐用，避免一次性垃圾。
3.  **连接大于分类 (Linking > Nesting)**：优先建立笔记间的双向链接，而非深层文件夹嵌套。
4.  **输出倒逼输入**：定期将笔记整理为文章或分享，在输出中深化理解。

## 🚀 快速开始

如果你也想基于此结构搭建自己的知识库：

1.  **安装 Obsidian**: 前往官网下载并安装最新版应用。
2.  **克隆仓库**: 将本仓库克隆到本地。
    ```bash
    git clone https://gitee.com/nllg2002/obsidian-markdown-nllg-note.git
    ```
3.  **打开保险库**: 使用 Obsidian 打开刚才克隆的文件夹。
4.  **安装插件**: 首次打开可能需要按照提示启用社区插件（确保已开启安全模式）。
5.  **自定义**: 根据需要修改 `.obsidian/snippets` 或模板文件。
