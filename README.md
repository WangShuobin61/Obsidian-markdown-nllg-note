## 📖 使用指南

### 基本操作

- 新建笔记：`Ctrl/Cmd + N`
    
- 搜索笔记：`Ctrl/Cmd + O`
    
- 创建链接：`[[笔记名称]]`
    
- 插入双链：`[[`后输入笔记名称
    
- 标签系统：`#标签名称`进行分类
    

### 笔记规范

1. **文件命名**：使用英文或拼音，避免特殊字符
    
2. **元数据**：每篇笔记顶部使用YAML frontmatter
    
3. **标题层级**：合理使用H1-H6，保持结构清晰
    
4. **链接引用**：及时建立相关笔记间的双向链接
